## Proporción de color

La correcta aplicación de proporción de colores garantiza una homogeneidad en todos los materiales de comunicación. Su uso puede ser aplicado con una predominancia de la paleta principal o de una predominancia de una paleta secundaria.

### Proporción con paleta principal predominante

El uso de la paleta principal en proporciones **60 / 30 / 10** es el ideal para la mayor cantidad de piezas de comunicaciones. Garantiza una consistencia y diferenciación de la marca. La mezcla con la paleta complementaria permite manejar diferentes niveles de lectura y jerarquía dentro de una pieza.

```
+60%  →  Color principal (Azul Vivo / Blanco)
 30%  →  Color complementario
 10%  →  Color de acento
```

### Proporción con paleta secundaria predominante

Utilizada para piezas digitales por su facilidad de lectura, los colores de la paleta digital y los acentos de la paleta complementaria se manejan con la proporción sugerida de **60 / 30 / 10**.

```
60%  →  Color de paleta digital (fondo neutro)
30%  →  Color de paleta principal
10%  →  Color de acento complementario
```

## Resumen: código de colores

```
/* Paleta Principal */
--color-azul-vivo:       #2D6DF6;   /* rgb(45, 109, 246)   */
--color-blanco-puro:     #FFFFFF;   /* rgb(255, 255, 255)  */

/* Paleta Complementaria */
--color-azul-sura:       #0033A0;   /* rgb(0, 51, 160)     */
--color-amarillo-alegre: #E3E829;   /* rgb(227, 232, 41)   */
--color-aqua:            #00AEC7;   /* rgb(0, 174, 199)    */
--color-gris:            #888B8D;   /* rgb(136, 139, 141)  */

/* Tonos Neutros */
--color-neutro-azul:     #81B1FF;   /* rgb(129, 177, 255)  */
--color-neutro-violeta:  #8A9CD3;   /* rgb(138, 156, 211)  */
--color-neutro-amarillo: #ECF0A1;   /* rgb(236, 240, 161)  */
--color-neutro-aqua:     #9BE1E9;   /* rgb(155, 225, 233)  */
--color-neutro-gris:     #B4B4B5;   /* rgb(180, 180, 181)  */

/* Fondos Digitales */
--fondo-azul-claro:      #DFEAFF;   /* rgb(220, 234, 255)  */
--fondo-gris-claro:      #E5E9EA;   /* rgb(229, 233, 234)  */
--fondo-amarillo-claro:  #F9FAE1;   /* rgb(249, 250, 225)  */
--fondo-aqua-claro:      #D5F5F8;   /* rgb(205, 246, 249)  */
--fondo-blanco-suave:    #F8F8F8;   /* rgb(248, 248, 248)  */
```
